Graph Tool

Graph Node UI for Unity UI Toolkit

A Curious Trove R&D project exploring graph interfaces that can be authored in the editor and still function at runtime.

Active R&D

Current Direction

Editor and runtime overlap

The goal is a graph UI foundation that does not fall apart once you leave editor-only tooling.

Custom graph surfaces

The toolkit is aimed at teams that need their own nodes, ports, inspectors, and interaction patterns rather than a fixed one-size-fits-all graph window.

Not publicly released yet

This page is intentionally a status page, not a launch page. Release details will appear here once the product scope is stable enough for broader use.
