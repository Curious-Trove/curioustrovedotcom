Getting Started with OctoShaper

Learn the basics of OctoShaper and create your first procedural building in Unity.

Introduction

Welcome to OctoShaper! In this tutorial, you'll learn how to set up OctoShaper in your Unity project and create your first procedural building. By the end, you'll understand the core concepts of node-based procedural generation.

Prerequisites

  • Unity 2021.3 or later installed
  • Basic understanding of Unity Editor
  • OctoShaper package downloaded

Step 1: Installation

First, let's install OctoShaper in your Unity project:

  1. Open your Unity project
  2. Go to Window → Package Manager
  3. Click the + button and select Add package from disk
  4. Navigate to the OctoShaper package and select package.json

Step 2: Creating Your First Node Graph

Now let's create a simple procedural building:

  1. Right-click in the Project window
  2. Select Create → OctoShaper → Node Graph
  3. Name it "MyFirstBuilding"
  4. Double-click to open the Node Graph Editor

Step 3: Adding Nodes

Let's add some basic nodes to create a simple building structure:

  • Right-click in the graph and select Add Node → Primitives → Box
  • Add a Transform node to position the box
  • Add an Output node to preview the result
  • Connect the nodes by clicking and dragging from output to input ports

Next Steps

Congratulations! You've created your first procedural building with OctoShaper. Here are some ideas to explore next:

  • Experiment with different primitive shapes
  • Add randomization to create variations
  • Try the Array node to create repeated patterns
  • Explore the Material nodes to add colors and textures
